## How to Run

1. **Clone the Repository**  
   ```bash
   git clone <repository-url>
   cd <project-folder>

2. **Set Up Players
Configure two players using any combination of the following player types:

-human  
-strategy1  
-strategy2  

3. **(Optional) Customize Rules
If you'd like to change the game rules, choose from these rule sets:

-reverse  
-fallen-ace  
-same  
-plus  

4. **Run the Game
Press the Run button and enjoy the game!
